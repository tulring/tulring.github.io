J-Link烧录

使用U-Boot下载bin文件

  使用U-Boot下载程序文件相对于EOP下载方式要快很多，但是要先使用EOP下载U-Boot程序到板内，U-Boot执行后，才可以使用其下载功能。
  开发板自带两个mini USB接口，其中一个是USB转串口，用于输入输出控制指令和显示数据。另外一个是USB Device接口，即USB从设备，用于下载程序文件。

建立串口通讯

   - 将开发板USB转串口端口连接到PC，安装PL2303驱动，注意此处驱动需要安装老版驱动，新版驱动不支持开发板设备。
   - 将串口转接线接入PC，识别到串口后，打开MobaXterm软件，选择Session –> Serial,然后选择对应的串口和波特率和setting选项。然后开发板调到nor启动，重启。在串口监控界面在倒计时结束前敲击键盘即可进入u-Boot界面。

建立文件传输通道

   在建立了串口通讯后，还要建立文件传输通道才可以下载文件到开发板。步骤如下：
   - 将USB device接口接到PC，然后打开Zadig软件，选择Options –>List All Device,然后选择SEC S3C2440，libusb-win32，点击Install Driver即可安装开发板USB Device驱动。
   - 在串口监控界面进入u-Boot界面，按n选择下载文件到nand后，打开DNW_100ASK.ee,显示USB：OK后下载.bin文件。将开发板调成NAND启动，重启即可执行下载的程序。

启动代码编写

  代码首先关闭看门狗，防止系统不断重启。然后设置堆栈，系统为NAND Flash启动时，系统上电会把NAND Flash内部4K内容复制到内部RAM中运行。因此设置为1024*4。随后设置跳转函数为main函数。系统便会调用.c文件中的main函数。

C程序代码编写

Makefile编写

   arm-linux-gcc -c -o start.o start.s 命令由 start.s编译生成 start.o文件
   arm-linux-gcc -c -o main.o main.c 命令由 main.c编译生成 main.o文件
   arm-linux-ld -Ttext 0 start.o main.o -o LED.elf 命令由 start.o和main.0链接生成 LED.elf文件
  arm-linux-objcopy -O binary -S LED.elf LED.bin 命令由 LED.elf转换生成 LED.bin文件

编译，下载，运行

  将以上文件拷贝到ubuntu下，执行make编译生成.bin文件。将.bin文件拷贝到PC下载到开发板。将开发板调到NAND Flash运行模式，重启即可看到LED闪烁。

S3C2440时钟

时钟树分析

时钟源选择

  S3C2440时钟源根据时钟源选择引脚OM[2.3]选择。正常我们将其接到地，即选择外部时钟源12MHz输入。

Lock Time设置

   FCLK由外部时钟源经过MPLL生成，在系统上电一段时间内，外部时钟源开始震荡并且提升到正常频率。系统复位启动，此时系统时钟FCLK为时钟源频率，在经过Lock Time后系统将时钟源切换到MPLL输出，从而提高系统运行频率。Lock Time由寄存器LOCKTIME(0x4C000000)设置，默认为0xffff ffff。我们设置成默认值即可。

设置分频系数比

  外部时钟源经过MPLL产生FCLK，经过UPLL产生UCLK，UCLK主要供USB使用。FCLK经过分频寄存器CLKDIVN(0x4C000014)中的分频因子 HDIVN，PDIVN产生HCLK和PCLK。
   - FCLK 供CPU使用
   - HCLK 供AHB总线使用
   - PCLK 供APB总线使用

  其中HDIVN和PDIVN设置参考下图：


  根据列表我们选择分频系数比例1:4:8。因此设置CLKDIVN(0x4C000014) = 0X5。

设置异步模式

  当HDIVN不为0时，需要设置CPU工作于异步模式，否则CPU的工作频率为HCLK所产生的频率。

UART配置过程

配置相应管脚功能

  S3C2440配置TXD0，RXD0分别接到GPH2，GPH3。因此需要配置H2，H3为TXD，RXD模式。使能内部上拉。

  如图所示，只需将GPHCON寄存器[5:4][7:6]设置为10，然后使能内部上拉，将GPHUP寄存器对应位设置为1。
  注意：设置时最好按位处理，其他位数据不变，防止改变其他设备配置。现将对应位&0，然后 | 设置数据。

选择时钟，设置收发模式

  S3C2440的UART有三个时钟源：PCLK，UEXTCLK，FCLK/n。主要控制寄存器为UCONn。CUON0[11:10]=00，选择PCLK 50MHz，[0:3]=0101选择收发为中断模式。
