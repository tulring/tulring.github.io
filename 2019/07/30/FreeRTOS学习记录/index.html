发表于 2019-07-30 | 分类于 STM32

FreeRTOS移植

源码获取

  正点原子开发板资料中包含 FreeRTOSv9.0.0.exe 文件，解压后即可得到源码。解压后的源码文件目录如下：

  • FreeRTOS
    • Demo （官方例程）
    • Source （源码）
    • License （许可证）
  • FreeRTOS-PLUS
      本次以STM32F103为基础移植FreeRTOS，所以需要在Demo文件夹内找到例程CORTEX_STM32F103_Keil，拷贝其中的FreeRTOSConfig.h文件到Source文件夹下。并将source ->portable下除MemMang，RVDS，keil文件外其他文件夹删除。此时Source文件夹目录如下：
  • Source
    • include
    • portable
      • keil
      • MemMang
        • heap_4.c
      • RVDS
        • ARM_CM3
          • port.c
          • portmacro.h
    • croutine.c
    • event_groups.c
    • FreeRTOSConfig.h
    • list.c
    • queue.c
    • tasks.c
    • timers.c

向工程中添加文件

  在代码工程目录下创建FreeRTOS文件夹，将Source文件夹下所有文件拷贝到FreeRTOS文件夹下。工程中新建FreeRTOS分组，添加如下八个文件到分组中：

  • croutine.c
  • event_groups.c
  • FreeRTOSConfig.h (FreeRTOS配置文件)
  • list.c
  • queue.c
  • tasks.c
  • timers.c
  • port.c
  • heap_4.c （内存管理方法）
